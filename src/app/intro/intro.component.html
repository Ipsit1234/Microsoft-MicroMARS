Problem Statements

There are two problem statements we have worked on:

One-Way Trip!

  • Shortest path traversal from one source to one destination while
    1. avoiding absolute obstacles that cannot be crossed
    2. crossing uneven surfaces that add weights to the graph, can be accessed by using the terrain option in the menu bar

Intermediate Stops!

  • Travelling multiple destinations chosen by the user having
    1. Unordered destinations, for the programme to decide the optimum path
    2. Ordered destinations, decided as per user preference
